Welcome to GitHub Pages.

El objetivo de este curso es familiarizarse con los lenguajes de programación y las herramientas que nos permitan desarrollar aplicaciones con tecnología web.

inicialmente sólo colocaremos un código propuesto para realizar la práctica, pero hemos de tener en cuenta que podemos utilizar variaciones.

Los documentos de las prácticas pueden editarse con cualquier programa de edición de texto y navegador

Designer Templates

Para realizar alguna de las prácticas propuestas se recomienda familiarizarse con herramientas para el diseño de Mockups y Diagramas de Flujo

Rather Drive Stick?

No se requiere conocimientos previos sobre los temas que van a tratarse pero si es recomendable estar familiarizado con algún lenguaje de programación a fin de asimilar mejor los conceptos generales.
